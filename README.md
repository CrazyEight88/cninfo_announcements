## 运行时间概述：
* 100个公司（十年跨度），下载花费1小时；分析花费0.5小时；需要空间50G；
* 6000个公司，大约下载花费3天；分析花费1.5天；需要空间3T;(当然可以尝试下载完直接分析，然后移除)
## 设计思路：
1. 公司code，获取公司页面；
2. 公司页面获取公告链接；
3. 公告链接获取pdf或html；
4. 对pdf或html进行词频统计；
